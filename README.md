Reviora (Restorasi Hutan)

Reviora adalah game edukasi bertema restorasi hutan yang dikembangkan menggunakan Unity Engine. Game ini mengangkat isu deforestasi dan mengajak pemain memulihkan lingkungan hutan yang terkontaminasi melalui gameplay bergaya cozy simulation.

Dikembangkan sebagai bagian dari program Studi Independen di PT Kinema Systrans Multimedia (Infinite Learning).

Gameplay:
Pemain membersihkan kontaminasi di area hutan menggunakan mekanisme vacuum, lalu menanam dan menyiram seed yang diperoleh hingga tumbuh menjadi pohon. Setelah seluruh area dipulihkan, peta bertransisi dari kondisi terkontaminasi menjadi hijau dan subur.

Mekanik utama:
- Vacuum kontaminasi
- Plant & spray seed
- Sistem pertumbuhan pohon
- Inventory seed sederhana
- Transisi map (fade out) saat restorasi selesai

Tools:
- Unity Engine
- C#
